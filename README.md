# Neha Chappal Creative Suite

An AI-powered marketing automation suite built for a real footwear retail business — generates product photography, branded images, and short marketing videos using Gemini Flash and Veo.

This wasn't a practice project. It's deployed and actively used by Neha Chappal Store to replace expensive agency work. I identified the problem, defined the requirements, built the solution, and measured the results.

We use the following tech stack:
- Google Gemini Flash for image generation and editing
- Google Veo for short video creation
- TypeScript + React for the interface
- Tailwind CSS for styling

## 🚀 Features

* **AI Product Photography:** Transforms basic product shots into professional studio-quality images with clean backgrounds and optimized lighting
* **Smart Watermarking:** Auto-applies brand logo and watermarks consistently across all generated assets
* **Marketing Video Generation:** Creates short promotional videos from product images and a description prompt
* **Batch Processing:** Generates content for an entire product catalog in one session — not just one-off images
* **Platform-Ready Output:** Images sized and formatted for Flipkart, Amazon, Meesho, and Instagram with one click

## Business Impact

* ⏱️ Content creation time: **2 days → 20 minutes** per product batch
* 💰 Eliminated ₹15,000+/month in agency photography costs
* 📈 Improved listing quality on e-commerce storefronts leading to better conversion rates
* ✅ Used in production — not a demo

## Setup

```bash
git clone https://github.com/yatinbhalla/Neha-Chappal-Creative-Suite.git
cd Neha-Chappal-Creative-Suite
npm install
echo "GEMINI_API_KEY=your_key_here" > .env.local
npm run dev
```
